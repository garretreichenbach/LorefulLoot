# LorefulLoot

**LorefulLoot** is a mod for StarMade that seamlessly integrates lore-friendly wreckages, derelict stations, and pirate ambushes into your galaxy's generation. 

It completely overhauls the static spawning logic and introduces a highly customizable JSON-based spawning and loot engine.

## Features
- **Dynamic Wreckage Generation:** Entities spawned via LorefulLoot automatically become wreckages with zero faction association and disabled cores, making them prime targets for salvage operations.
- **Battle Debris:** Ships destroyed during combat naturally become salvageable wreckages instead of disappearing.
- **Weighted Loot Pools:** Customize exactly what items spawn in wreckages, complete with random min/max bounds and RPG-style weighted drop chances.
- **JSON Configuration:** A lightweight JSON engine replaces the old complicated Lua scripts, making it trivial to add your own blueprints and custom encounters.
